Preface

This section of the FAQ deals with topics specific to the OS/2 system and related/derivative platforms.

Questions
  • Are there Rexx scripts to run Tomcat on OS/2?
  • Are there other scripts to run Tomcat on OS/2?
Answers
Are there Rexx scripts to run Tomcat on OS/2?
Yes, at NMSU.edu. If the above link doesn't work, try a search on hobbes.nmsu.edu, as the above link is specific to Tomcat 5.0.27.

Are there other scripts to run Tomcat on OS/2?
Yes, at NetBeans.org. Note again that the above scripts may be specific to a Tomcat version or branch.
